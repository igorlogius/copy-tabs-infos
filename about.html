Get tab information in a user defined custom format string usage:
  1. click the toolbar icon
  2. (optional) create the format strings you want
  3. select the format string you want to use (radio button)
  4. select the scope "active", "selected" or "window"
  5. (optional) further edit the otuput in the textarea
  6. click the "Copy output" button to copy the content of the textarea
